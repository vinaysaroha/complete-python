# int: Whole number such as 3,300,200
# float: number with decimal points 2.4,5.6
# str: Order sequence of character  "hello 'sam' how are you"
# list: Order sequence of object [1,3,6,7]
# dict: Unorder key value pair {'key1': 'value1','key2':'value2'}
# tup: Order immutable sequence of object('a',1,5,'txt')
# set : Unorder collection of unique objects('a','b')
# bool : logical value indicates True or false
